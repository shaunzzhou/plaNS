## How does our automatic duty roster scheduling app work?

#### Input
Users are able to input the dates that they are not available for duty.
<br/>
<img src="https://github.com/shaunzzhou/plaNS/assets/107353043/0a61523d-8525-4610-a4fa-fc53a2e4bf1c" width=30% height=30%>
<br/>

#### Roster Authentication
The administrators can generate a roster schedule with a unique ID stored in the database.
<br/>
<img src="https://github.com/shaunzzhou/plaNS/assets/107353043/e08a35eb-85d2-4a95-b949-34a60a0f5454" width=30% height=30%>
<br/>

#### Generate Schedule using Algorithms
A schedule will be generated using breadth first search algorithms and balanced using a fairness algorithm.
